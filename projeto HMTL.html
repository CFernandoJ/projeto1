Historia do HTML.

O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente.

Ele publicou a primeira versão do HTML em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos (modificadores de tags).

O maior upgrade da linguagem foi o lançamento do HTML5 em 2014. Diversas novas tags semânticas foram adicionadas que revelam o significado do seu próprio conteúdo.

Como Funciona.

Reconhecidos por, normalmente, estarem posicionados entre os sinais de chevron, “< >”, os comandos HTML definem toda a estrutura de uma página HTML.

Eles são os responsáveis por dizer para os navegadores onde os elementos começam e terminam. Assim, junto aos seus atributos que, por sua vez, são os responsáveis pelas características desses elementos, as páginas da web são formadas.

O que são tags.

As tags HTML são elementos usados para marcar e formatar os conteúdos: textual ou visual em documentos HTML.

Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Por exemplo: a tag de parágrafo é escrita como "p". Já para negritar pode ser usada a tag "Strong" ou "b" no início e para finalizar "/strong" ou "/b".

O que são comandos HTML.

Um comando HTML é um objeto identificável dentro de um documento, e pode resultar em uma imagem, uma tabela, ou qualquer outro elemento visível.

Como aprender HTML.

Não há dúvidas de que o aprendizado de HTML é bastante valioso para a carreira em Tecnologia da Informação e pode ajudar a impulsionar a sua vida profissional.

Além disso, segundo o site Tech Crunch, o Tik Tok, uma das redes sociais em maior crescimento no momento, anunciou que está testando mini jogos dentro da plataforma desenvolvidos com HTML5.
